# TUI-NoteTaker
Outil utilisable dans le terminal pour gérer une bibliographie et la prise de note

## Objectifs
- Centraliser la gestion de la prise de note et de l’export des données dans le cadre de recherches
- Créer une application CLI/TUI avec PHP
- Me faire la main sur la gestion d’un monorepo

## Pourquoi ?
Dans le cadre des recherches pour une présentation, je ne trouvais pas d’outils qui me conviennent (à part vaguement [Raindrop.io](https://raindrop.io/). Ayant un TDAH, devoir passer d’une application à une autre, ou pire, d’un onglet à un autre demande beaucoup d’énergie pour rester concentrée. 
Le terminal est un moyen pour moi de ne pas m’éparpiller, et j’apprécie de limiter l’usage de la souris. 

J’ai voulu profiter de ce manque pour créer un outil qui me conviendrait.
Et ce faisant, pourquoi pas me faire la main avec l’aspect fullstack, apprendre comment gérer un monorepo, continuer de mettre en œuvre l’approche TDD, découvrir de nouvelles façons de créer des interfaces, et découpler mes applications. 

## MVP
- Une ligne de commande pour ajouter des liens, des citations et des notes
- Un export au format markdown

## Et ensuite ?
- Une application TUI pour gérer les projets, les catégories, les liens
- Pouvoir utiliser le contenu du presse-papier directement
- Créer une application web
