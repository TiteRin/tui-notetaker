# ✅ Roadmap & Checklist

Cette page liste les étapes à réaliser pour développer **MonApp**, une application locale packagée en binaire unique avec **FrankenPHP**, **API Platform (Laravel)** et un **monorepo géré par moonrepo**.

---

## Monorepo & outillage

- [ ] Initialiser le repo avec **moonrepo**
- [ ] Définir les workspaces :
  - `apps/api`
  - `apps/cli`
  - `apps/tui`
  - `libs/domain`
  - `libs/infra`

---

## Backend API

- [ ] Installer **Laravel**
- [ ] Ajouter **API Platform (compat Laravel)**
- [ ] Configurer **SQLite** par défaut
- [ ] Créer un modèle simple (`Book`)

---

## CLI

- [ ] Ajouter un CLI (**Symfony Console** ou **Node Commander.js**)
- [ ] Commande `books:list` consommant l’API (HTTP)
- [ ] Vérification & lancement auto du daemon (**lazy mode**)

---

## TUI

- [ ] Ajouter un TUI (**Ink.js** ou **php-school/cli-menu**)
- [ ] Naviguer dans la liste des livres

---

## Lazy Daemon

- [ ] Script wrapper `monapp`
- [ ] Vérifier si un port est ouvert → sinon lancer FrankenPHP
- [ ] Ajouter un watchdog/timeout pour arrêt après X minutes d’inactivité

---

## Packaging

- [ ] Créer un `Caddyfile` pour FrankenPHP
- [ ] Build binaire avec `frankenphp build`
- [ ] Tester sous Linux/MacOS

---

## Persistance

- [ ] Configurer migrations SQLite
- [ ] Stockage local dans `data/`
